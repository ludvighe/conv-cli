# Conv

Command line tool to convert between scales.

---

## --help, -h

```
-------------------------------------------------
Conv - a converter to convert between unit scales
-------------------------------------------------
Temperature scales:
  - °C  Celsius     "c" | "celsius"
  - °F  Farenheit   "f" | "farenheit"
  - K   Kelvin      "k" | "kelvin"
  - °R  Rankine     "r" | "rankine"
  - °Ré Réaumur     "re" | "ré" | "reaumur" | "réaumur"

Usage: conv [OPTIONS] <VALUE> <FROM> <TO>

Arguments:
  <VALUE>  The numerical value to be converted
  <FROM>   The scale from which to convert
  <TO>     The scale to which the value should be converted

Options:
  -v, --verbose  
  -h, --help     Print help
  -V, --version  Print version

```
